# Diagrams

This directory contains PlantUML sources for the main application architecture and critical workflows.

Files:
- `functional-architecture.puml`: high-level functional architecture of the AGUI app, backend, MCP layer, and cloud integrations.
- `login-sequence.puml`: AWS CLI login and identity-refresh sequence.
- `terraform-infra-sequence.puml`: Terraform-based infrastructure setup flow through chat, MCP, and Terraform.
- `maker-checker-sequence.puml`: approval and execution workflow for gated infrastructure changes.
